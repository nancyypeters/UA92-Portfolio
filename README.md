# UA92 Portfolio

Coursework and projects from Level 5 Computer Science at UA92, Manchester.

## Contents

### 🗄️ Databases (UA92-234)
SQL database design and development for Spark Academy, a fictional creative arts institution.

- `script1.sql` — Database schema: 10 tables covering Campus, Students, Courses, Enrolments, and Certificates
- `script2.sql` — Database population with sample data across all tables

**Tools:** SQLite · SQLiteStudio

### ☁️ Cloud Computing (UA92-233)
Automated tournament management system built for 92 Esports League using Microsoft Power Platform.

- 4 Power Automate flows covering registration, scheduling, results, and reminders
- Part of a group project building an end-to-end tournament management system

**Tools:** Power Automate · SharePoint · Microsoft Forms

### 🏫 School Portal
A PHP-based school management system with role-based access for admins and teachers.

- Admin and teacher dashboards
- Pupil, parent, and teacher registration and management
- Class management and password handling

**Tools:** PHP · MySQL

## Languages & Tools

SQL · PHP · Power Automate · SharePoint · Microsoft Forms · SQLite
